项目介绍
本项目是一款轻量级的桌面端文件批量处理工具，支持对指定目录下的文件进行批量重命名、格式转换、大小压缩等操作，旨在帮助用户高效处理大量文件，提升办公与个人文件管理的效率。工具采用模块化设计，支持自定义处理规则，适配多种常见文件格式，操作简单易上手，无需复杂的技术背景即可快速上手使用。
使用方法
配置处理规则：打开工具后，在「规则设置」界面选择需要执行的操作（如批量重命名、图片压缩等），并根据需求自定义参数（如重命名前缀、压缩质量等）。
执行批量处理：选择需要处理的文件目录，点击「开始处理」按钮，工具将自动按照预设规则完成所有文件的批量操作，处理完成后会生成详细的操作日志供用户核对。
